# SCONS Project

## Descrição

Este é um projeto de landing page para o **SCONS (Soares Campos Optimized Navigation System)**, uma plataforma conceitual unificada para gestão de aplicações empresariais. O projeto foi desenvolvido como uma demonstração de habilidades em desenvolvimento web moderno, focando em uma interface elegante e interativa.

**Atenção:** Este é um projeto demonstrativo e não representa uma aplicação funcional completa. É uma vitrine de tecnologias e design para portfólio, mostrando como seria uma página inicial para um sistema de gestão empresarial.

## Funcionalidades

- **Seção Hero:** Apresentação da plataforma com animações e estatísticas
- **Recursos Principais:** Destaque para funcionalidades como login centralizado, acesso hierárquico e previews de aplicações
- **Previews de Aplicações:** Carrossel interativo mostrando 6 aplicações conceituais:
  - Dashboard Executivo (SCONS Analytics)
  - Sistema de RH (SCONS Gestão)
  - Portal Financeiro (SCONS Finanças)
  - CRM Vendas (SCONS Vendas)
  - Sistema de Estoque (SCONS Operações)
  - Portal de Projetos (SCONS Projetos)
- **Animações:** Fundo animado com elementos flutuantes e transições suaves
- **Design Responsivo:** Otimizado para desktop e mobile

## Tecnologias Utilizadas

- **Next.js 15.5.3** - Framework React para aplicações web
- **React 19.1.0** - Biblioteca para interfaces de usuário
- **TypeScript** - Superset JavaScript com tipagem estática
- **Tailwind CSS 4** - Framework CSS utilitário
- **Framer Motion** - Biblioteca para animações
- **Lucide React** - Ícones SVG
- **ESLint** - Linting para qualidade de código

## Design e UX

O projeto utiliza um tema dark com gradientes sutis, animações fluidas e efeitos de blur para criar uma experiência moderna e profissional. As animações são implementadas com Framer Motion para performance otimizada.
